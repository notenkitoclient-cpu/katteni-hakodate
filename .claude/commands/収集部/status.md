# 【収集部】収集状況を確認する

収集部が管理する全ソースの現状を横断的に確認する。

## 確認内容

以下をすべて調べて報告してください：

1. **SNSキュー** (`src/content/sns-queue/`)
   - 未処理画像の件数（`_done_` プレフィックスなし）
   - 処理済み画像の件数

2. **ニュース蓄積数** (`src/content/news/`)
   - 総件数
   - 直近7日間の件数
   - type別内訳（開店/閉店/イベント/目撃情報/その他）

3. **記事キュー** (`src/content/article-queue.txt`)
   - 現在のキュー内容を全行表示

4. **スプレッドシート処理済みID** (`src/data/sheets-processed-ids.json`)
   - 処理済みID件数
   - 最後に処理した日付（ファイルの更新日時）

5. **直近の収集ワークフロー実行結果**（わかる範囲で）

報告は箇条書きで簡潔に。問題があれば対処方法も提示する。
