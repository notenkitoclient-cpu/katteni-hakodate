# カッテニハコダテ サイト運用戦略計画書

**バージョン**: 1.0
**策定日**: 2026年3月31日
**対象期間**: フェーズ1（3ヶ月）〜 フェーズ3（12ヶ月以降）

---

## エグゼクティブサマリー

「カッテニハコダテ」は、函館に根ざしたコミュニティ主導の情報プラットフォームである。
サイコーニセコを参照モデルとしつつ、枚方つーしん・Visit Iceland・瀬戸内DMOの成功要素を統合し、**「住民が毎日使う生活インフラ」×「観光客の旅情を誘う入口」** という二軸で函館の地域経済を牽引することを目的とする。

---

## 第1章 コンセプト設計

### 1-1. ブランドコンセプト

| 要素 | 内容 |
|------|------|
| **サイト名** | カッテニハコダテ |
| **コンセプト** | 「かってに（勝手に／好き勝手に）函館を発信する」地元目線の情報プラットフォーム |
| **キャッチコピー案** | 「知らなかった、ハコダテがある。」 |
| **ポジショニング** | 観光ガイドでも行政サイトでもない「地元民の口コミ」が集まる場所 |
| **参照モデル** | サイコーニセコ（コミュニティ型）＋ 枚方つーしん（即時性・実用性） |

### 1-2. 二軸ターゲット戦略

```
┌─────────────────────────────────────────┐
│           内部ビュー（住民向け）          │
│  函館市民・近郊住民・UIターン検討者       │
│  → 日常性・利便性・即時性を最大化        │
├─────────────────────────────────────────┤
│           外部ビュー（観光客向け）        │
│  道外旅行者・インバウンド・移住検討者     │
│  → 非日常のストーリー・旅情を最大化      │
└─────────────────────────────────────────┘
```

**住民が「毎日見る」→ 観光客が「信頼する」** という好循環を設計する。
（枚方つーしんが市外からの取材を受けるほど信頼されたのも、住民の日常利用が土台）

## 第2章 コンテンツ戦略

### 2-1. カテゴリ構成

| カテゴリ | 内部ビュー（住民） | 外部ビュー（観光客） |
|---------|------------------|-------------------|
| **グルメ** | 新規開店・閉店情報、地元民が通う店 | 函館朝市・海鮮・ご当地グルメ深掘り |
| **観光・スポット** | 穴場・季節の変化・週末の過ごし方 | 王道スポット＋地元民しか知らない場所 |
| **暮らし** | 子育て環境・移住情報・行政サービス | 移住体験記・生活コスト・リモートワーク事情 |
| **イベント** | リアルタイム混雑情報・現地レポ | イベントカレンダー・季節のハイライト |
| **人・ビジネス** | 地元の人・店の物語（ネイティブ広告） | 函館のクリエイター・生産者・職人 |
| **特産品** | 地元産品の購入情報・ふるさと納税 | お土産・産直・オンライン購入 |

### 2-2. コンテンツの鉄板企画（枚方つーしん分析より）

住民が必ず反応する「日常の変化」を押さえることが最優先。

1. **開店・閉店速報** — 街を歩いて取材。「もう閉まった?」を先に答える
2. **現地リアルタイムレポ** — 混雑状況、行列の長さ、今日の景色
3. **「知らなかった函館」シリーズ** — 地元民でも驚くニッチ情報
4. **住民インタビュー / Saiko Makers的企画** — 函館で働く・暮らす人の物語
5. **季節コンテンツ** — 函館の四季（桜・函館港まつり・紅葉・雪景色・イカ漁）
6. **緊急・防災情報** — 台風・大雪時の現地情報（社会的価値を高める）

### 2-3. アイスランド式「人々をメディアにする」戦略

サイコーニセコ同様、**コミッティメンバー制度**を函館版で実装する。

```
函館コミッティメンバー（仮称：ハコダテ仕掛け人）
├── 募集条件：函館が好き、情報を発信したい
├── 役割：自分の専門領域（飲食・アート・スポーツ等）で記事を投稿
├── 特典：名前・プロフィール掲載、地元ネットワーク構築
└── 目標：初年度10名 → 3年で30名体制
```

**Iceland Hour的キャンペーン案（函館版）**：
「ハコダテの好きな場所・食べ物を一斉投稿しよう」→ SNSで拡散 → サイトに集約

## 第3章 UI/UX設計指針

### 3-1. デザイン原則

| 原則 | 内容 | 参考事例 |
|------|------|---------|
| **モバイルファースト** | 旅行者の60%以上がスマホで検索 | Travel Oregon |
| **感情ナビゲーション** | 「今日の気分」からコンテンツを選べる | Travel Oregon |
| **季節変化するデザイン** | 函館の四季を視覚的に表現 | サイコーニセコ |
| **情報の即時性** | 今日・今週の情報が最前面に | 枚方つーしん |
| **人の顔が見える** | ライターやコミッティの顔写真を前面に | サイコーニセコ |

### 3-2. サイト構成（ページ設計）

```
TOP
├── 今日の函館（リアルタイム情報・PickUp）
├── グルメ
├── 観光・スポット
├── 暮らし・移住
├── イベント
├── 人・ビジネス
├── 特産品
├── ハコダテ仕掛け人（メンバー紹介）
├── カッテニ投稿（UGCフォーム）
├── 函館について（サイトコンセプト）
└── お問い合わせ・メンバー募集
```

### 3-3. 技術スタック推奨

| 要素 | 推奨 | 理由 |
|------|------|------|
| **CMS** | WordPress | 運用コストが低く、SEOプラグイン豊富 |
| **フロントエンド** | Elementor or Kadence | ノーコードで季節変化デザインを実装 |
| **ホスティング** | さくらインターネット / Xserver | 国内レイテンシ、コスト最適 |
| **CDN** | Cloudflare（無料プラン） | 画像配信の高速化 |
| **分析** | Google Analytics 4 + Search Console | SEO・UXの改善サイクル |
| **SNS連携** | Instagram・X（Twitter）・LINE公式 | 函館ターゲットの主要チャネル |

Google検索の46%が地域的意図を持つ。スマホ検索の88%が1週間以内に来店する。

### 4-1. キーワード設計の柱

```
【住民向けキーワード群】
「函館 新店 2026」「函館 子育て 環境」「函館 週末 おすすめ」
「函館 移住 リアル」「函館 ランチ おすすめ」「函館 カフェ 穴場」

【観光客向けキーワード群】
「函館 観光 モデルコース」「函館 朝市 おすすめ」「函館 夜景 穴場」
「函館 グルメ 地元民」「函館 旅行 準備」「函館 イカ 料理」

【移住促進キーワード群】
「函館 移住 メリット」「函館 リモートワーク」「函館 家賃 相場」
```

### 4-2. コンテンツSEOの実装

- **地域特化の詳細記事を蓄積**（月4〜8本のペース）
- **Googleビジネスプロフィール最適化**（サイト自体のブランド認知）
- **内部リンク設計**（「海鮮グルメ → 函館朝市ガイド → アクセス方法」の導線）
- **構造化データ（Schema.org）** でレビュー・イベント情報をリッチリザルト対応

## 第6章 収益モデル

枚方つーしんが「3年かけて収益化に成功した」ように、最初は信頼構築を優先する。

### 6-1. 収益化ロードマップ

```
フェーズ1（0〜6ヶ月）: 信頼構築期
└── 収益なし。コンテンツ品質と住民エンゲージメントに集中

フェーズ2（6〜12ヶ月）: 収益化開始期
├── ネイティブ広告（記事広告）: 地元企業の魅力を記者目線で紹介
├── バナー広告: 地元企業・観光施設
└── イベントスポンサー: 地域イベントの協賛枠

フェーズ3（12ヶ月以降）: 拡大期
├── プレミアム掲載プラン（店舗・観光施設の強調表示）
├── 移住促進コンテンツの行政連携（函館市・道南各町との協定）
├── 地元企業のSNS運用代行・デジタルマーケティング支援
└── 物販・ふるさと納税アフィリエイト
```

### 6-2. ネイティブ広告の設計（最重要）

枚方つーしんの最大の差別化要素は「記事広告の質」。

- 地元ライター（コミッティメンバー）が実際に取材・体験して執筆
- 「PR」表記は明記しつつ、読者が読みたいと思える記事品質を維持
- 料金の目安（初期）：1記事 3〜5万円（函館の市場規模に合わせて設定）

## 第8章 運用体制

### 8-1. 初期フェーズの体制（少人数スタート）

```
編集長（1名）
├── コンテンツ企画・編集・SNS運用
├── コミッティメンバーのマネジメント
└── 広告営業（兼任）

コミッティメンバー（3〜5名）
└── 各専門領域（飲食・アウトドア・子育て・アート・移住等）で記事投稿

外部協力
├── カメラマン（フリーランス、必要時に依頼）
└── Web制作者（初期構築のみ）
```

### 8-2. 「足で稼ぐ」ことの重要性

枚方つーしんが「必死で街を歩き回った」ように、デジタルの信頼はオフラインの取材が支える。

- **週2回以上**: 函館市内をフィールドワーク（開店・閉店・イベント確認）
- **月1回以上**: 地元事業者インタビュー
- **季節ごと**: 函館の旬の風景・食材の現地取材

## 第9章 函館固有の強みと活用戦略

### 9-1. 函館だけが持つコンテンツ資産

| 資産 | コンテンツ展開の可能性 |
|------|---------------------|
| **函館夜景（世界三大夜景）** | 季節・天候別の夜景ガイド・ベストスポット |
| **函館朝市** | 地元民が教える「本当においしい朝ごはん」 |
| **イカ漁・海産物** | 漁師インタビュー・旬カレンダー |
| **歴史的建造物（元町・ベイエリア）** | 建物の物語・知られざる歴史 |
| **函館山** | 四季折々の植生・登山ルート地元民版 |
| **湯の川温泉** | 地元民が使う銭湯・日帰り温泉 |
| **道南のローカル食文化** | ラッキーピエロ・ハセガワストア・ガラナ |

### 9-2. 函館の課題をコンテンツで解決する

地域メディアの社会的価値は「課題解決」にある：

- **人口減少・移住促進**: 「函館で働く・暮らす」シリーズで移住ハードルを下げる
- **観光の質向上**: 「地元民が本当に行く店」を発信し、オーバーツーリズムを分散
- **地元企業の発信力強化**: デジタル苦手な店主の魅力をコミッティが代わりに発信

## まとめ：「カッテニハコダテ」が目指す社会的インパクト

> 枚方つーしんが「家族の会話のきっかけ」になり、
> Jimo-coroが「移住転職者数300%増」を実現したように、
> カッテニハコダテは函館のアイデンティティを再構築し、
> 物理的な移動（観光・移住）を誘発する強力な動機付けとなる。

**3年後のビジョン：**
- 函館市民が毎日チェックする「生活インフラ」になる
- 「函館に行きたい」と思わせる観光コンテンツの発信地になる
- 函館への移住者を年間100人単位で増やすきっかけになる
